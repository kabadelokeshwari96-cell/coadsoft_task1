# 📝 To-Do List Application

A simple command-line To-Do List application built with Python.  
This project helps you manage tasks efficiently by adding, viewing, updating, and deleting them.

---

## 🚀 Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Save tasks persistently (using a file/database)
- Simple and beginner-friendly code structure

---
